Quality Assurance
제품이 일정 수준의 품질을 가질 수 있도록 출시전에 테스트 및 검수작업을 하는 것

CPM (Cost Per Mille)
광고비용을 측정하는 모델
1000회 광고를 노출시키는데 사용된 비용을 의미함

dlc
게임이 발매된 이후 추가적으로 받을 수 있는 컨텐츠
라이브서비스 게임에서는 주로 쓰지 않는 표현
콘솔, 패키지 게임 시장에서 주로 사용되는 용어

재미로 숨겨놓은 메시지나 기능
게임 진행에 큰 영향을 미치지 않아야 함

CPA (Cost Per Action)
온라인 또는 모바일 환경에서 시행되는 직접반응 광고모델 중 하나
타겟이 원하는 행동을 취할 때마다 광고비를 지급하는것

CPI (Cost Per Install)
CPA의 하위 개념, 옛날에는 많이 썼는데 요즘은 사장되는 방식

CDN (Contents Delivery Network)
인터넷 환경에서 게임 클라이언트나 콘텐츠를 사용자에게 전달하기 위해 서버에 데이터를 저장해 전달하는 시스템(분산서버를 사용하는 경우가 다수)

어느 부분까지 클라이언트에 포함시키고
어느부분까지 서버에서 관리할것인가

IDC (Internet Data Center)
서버가 (물리적으로) 모여있는 곳
주로 서버를 대여해주는 기업이 갖추고있음

NCP
사전적 의미는 플레이어를 제외한 모든 캐릭터를 의미하지만
최근에는 플레이어를 제외한 아군 캐릭터를 지칭하는 용도로 주로사용됨

MAU (Monthly Active Users)
한달동안 서비스를 이용한 순수한 이용자 수 지표

DAU(Daily Active Users)
MAU의 하루단위 버젼

CC (Crowd Control)
상대방을 원하는대로 움직이기 어려운 상태로 만드는 효과나 그 기술

OP (OverPowered, Opening)
1. 게임에서 밸런스를 해칠정도로 강력한 성능을 가진것
2. 오프닝

pvp 와 pk
플레이어간 합의된 상태인가, 아닌가를 기준으로 나눠서 사용함

PVR, RVR (Realm VS Reaml)
시중에 통용되는 용어는 아니지만 다대다 전투를 지칭할 때 의미함
기획분야에서 구분해서 사용하는 경우가 많음

레이드(Raid)
다수의 인원이(일반적으로 한파티 이상의 규모) 모여서 던전이나 보스몬스터를 공략하는 컨텐츠

PVE (Player VS Environment)

Billing
결제 시스템
게임에서는 전자결제를 위한 시스템을 통틀어서 이야기함
(Electronic Billing System)

로그(LOG)
컴퓨터나 서버 등에서 여러정보를 기록으로써 남기는 것

CBT (Closed Beta Test)
비공개 베타 테스트
게임을 정식으로 런칭하기전에 게임의 여러 문제점을 찾거나 시장성, 재미 등을 검증하기 위해 진행함

오픈월드
높은 자유도를 기반으로 플레이어의 제약이 거의 없는 게임
    ~
심리스(Seamless)
지역과 지역 사이의 로딩이 없음

높은 퀄리티의 오픈월드를 만들기 위해 심리스 기술이 필수적이며, 오픈월드를 심리스 월드라고 부르는 경우도 있음



Regeneration
소비되는 자원이 시간이 지남에따라 자연적으로 회복되는 것

Respwan
어떤 오브젝트가 필드에 주기적으로 생성되는 것

BOT
멀티플레이 게임 혹은 멀티플레이 게임 모드에서 등장하는 인공지능 플레이어

DOT (Damage over Time)
일정시간에 걸쳐서 타깃에게 데미지를 주는 것

ARPU (Average Revenue per User)
게임에 가입한 유저 1명당 평균 결제금액
부분유료화에서 굉장히 중요한 지표

ARPPU (Average Revenue per paid User)
게임에서 결제 경험이 있는 유저 1명당 평균 결제금액
표준기간이 정의되어 같이 사용됨

채널링 (Channeling)
이미 서비스중인 온라인 게임을 다른 제휴사의 유저들이 접속 할 수 있도록 하는 서비스

renewal
기존의 기획은 남겨두되 사용자 중짐의 편의를 강조해서 게임을 대대적으로 개선하는 작업을 뜻함

Roaming
rpg - 설정된 구역을 반복적으로 돌아다니는 몬스터
aos - 다른 플레이어의 라인으로 넘어가 해당 플레이어를 돕는 플레이

Looting
대상을 쓰러뜨리고나서 대상이 가지고있던 물건 또는 떨어뜨리는 물건을 가져가는것

ADD
다양한 요인으로 인하여 계획되지 않은 몬스터가 전투에 끼어드는 상황

스트레스 테스트
게임의 서버 수용력, 다수의 유저가 일제히 접속했을 때, 게임이 원활하게 돌아가는지 테스트하는 것

Active User
실제로 일정한 기간동안 게임이나 컨텐츠를 이용한 총 사용자

관통
1. 일정한 방향성(벡터)을 가진 공격이 적을 뚫고 지나가는것
2. 내성이나 방어도 등을 무시하고 공격을 가하는 것

P2P (Pay to Play) - peer to peer랑 다름
일정 금액을 지불해야 게임을 할 수있는 시스템
기존의 네트워크 용어랑 헷갈려서 b2p(Buy to Play)로 대체해서 쓰기도 함

F2P (Free to Play)
게임자체는 무료, 부분유료화서비스

Resist
특정 공격이나 상태이상 효과에 대한 내성수치

Stack
게임에서 사용한 스킬이나 착용한 아이템의 효과가 중첩되는것

Object
접촉할 수 있으며, 감각을 통해 파악할 수 있는 것 또는 상호작용 가능한 모든 것, 지형이 아닌 모든 설치물 등의 의미를 가지고있다.

Frames
액션게임에서 모든 동작에 들어가는 일련의 시간을 수량화한 최소단위

리텐션(잔존률) = 사용자 만족도
일반적으로 이탈인원들이 많을 때 관련 작업을 함

GQA (Game dev QA)
개발QA로 기획서를 검토하여 테스트케이스를 만들고 프로그램팀에 넘기고, 프로그램팀에서 구현한 기능을 확인하고, 패치노트를 작성하며, 버젼관리를 함


PQA (Publishing QA)
대규모의 테스트케이스를 만들고 전체 검수를 진행함. 주로 테스트 서버에서 테스트가 이루어짐

SQA (Service QA)
운영QA로 유저관점에서 진행된다. 패치노트에 적힌 내용이 실제로 적용되었는지 확인함


TQA (Technical QA)
기획서에 예정된 내용대로 구현되었는지 기술적 내용을 테스트한다.

FQA (Fun QA)
게임에 반영된 컨텐츠 자체를 테스트한다. 유저적 관점에 초점을 맞추어 재미, 흥미유발 위주의 테스트를 함

CQA (Critical QA)
게임에 심각한 영양을 주는 문제(예를들면 템복사)를 전문으로 다룬다.

DQA (Data analysis QA)
게임 로그분석, 게임내 유저들의 행동패턴을 분석함, 유저들의 행동이 기획의도대로 되지않는다면, 그 이유를 분석함


TS (Time spent)
1인당 1일 플레이타임



MMORPG (Massive Multy Online Role Playing Game)
대규모 다중접속 온라인 롤 플레임 게임
유저 관점에서 볼 때 여러명의 사용자와 함께 같은게임을 플레이 한다는 점이 핵심이다.
타인과의 비교에서 얻게되는 우월감과 자기과시, 타인과의 협력을 통한 성취감과 소속감, 성장이나 목표달성을 통한 자기만족 등이 중요한 가치라고 볼 수 있다.

FPS (First person shooter) 1인칭 슈팅 게임
3차원의 공간을 게임상의 캐릭터 시점으로 누비며 적을 발사무기로 공격하는 게임장르 시점이 플레이어캐릭터 내부에 있는것이 특징
시점만 1인칭이면 fps인가? 발사무기를 사용하지 않으면 fps가 아닌가? 등 사전적인 의미의 범주가 무너지고있다.

TPS (Third person shooter) 3인칭 슈팅 게임
fps와 유사하나 시점이 플레이어캐릭터의 외부에 있는것이 차이점이다. 대부분 캐릭터의 머리위~후면에 위치해있다.

레이싱
경쟁상대와 스피드를 겨루는 게임장르
모터스포츠 장르인 자동차와 오토바이 경주를 게임화한 것이 가장 흔하다.

Shooting Game (STG)
적의 공격을 피하며 무기를 쏘는 게임의 총칭
필수 3요소
1. 플레이어
2. 적
3. 자신의 무장

AOS (Aeon of Strife)
Eul이 제작한 워3 유즈맵 도타로 완성된 장르
플레이어가 캐릭터를 선택하여 캐릭터를 강화시켜 상대방의 진영을 파괴하는 실시간 공성전
최근에는 MOBA라는 이름으로 불리기도 함

RTS (Real Time Strategy)
실시간 전략게임. 국내에서는 전략 시뮬레이션이라고 통용되어있지만 시뮬레이션적 요소가 있는 게임은 손에 꼽을정도.

Roguelike
게임 Rogue의 특징과 시스템을 모방/차용하여 만든 게임들의 총칭

대전액션
플레이어가 서로 대결을 하는 게임장르
격투를 게임으로 옮긴 장르

Simulation Game
모의 시험 게임
현실과 닮아있는 환경을 구성하는것이 핵심

Sandbox
모래놀이터에서 모래로 자유롭게 무엇인가를 만들어 낼 수 있는 특징을 따온것
높은자유도와 다량의 리소스를 제공하는것이 핵심과제

Puzzle game
퍼즐 = 어려운 문제, 깊이 생각하게 만드는 문제
퍼즐게임을 퍼즐을 푸는것으로, 어떠한 규칙내에서 정해진 행위를 통해 조건을 완료하는 단순한(변수가 적다는 의미) 게임
할만한 난이도를 제시하는것이 중요함

TCG (Trading Card Game) 최근에는 사전적 의미가 많이 엇나가서 세간에서는 CCG로 불리우기도 함
규칙에 따라 자신만의 덱을 만들어 상대와 대전하는 게임

rhythm game
플레이어가 리듬이나 음악에 맞추어 조작하거나 동작을 취하는게임
진행에 있어서 음악이나 리듬이 주 요소가 되는 게임장르
일본 = 음악게임
중국 = 음악유희, 음악절주유희
