
shot : 한번의 테이크를 통해 촬영된 장면
take : 카메라 작동스위치가 한번 기능하여 촬영된 연속적인 화면단위(쇼트의 기술적인 의미)
cut : 여러의미로 쓰임, 쇼트와 동일한 개념으로 쓰이거나, 잘라내는 의미, 장비의 정지
scene : 동실장소, 동일시간내 여러 각도에서 촬영된 쇼트들이 모여 만들어진 하나의 장면
sequence : 씬들의 집합의 최소단위
plot : 이야기의 인과관계
cliche : 상투적인표현


beat : 행동/반응 이라는 행위의 교환

사건은 구성되는것 : 배경과 이미지, 행동과 대화
사건이란 변화를 야기하는 것 : 사건을 통해 등장인물이 변화됨
이야기의 구조를 통해서 관객들들로하여금 주인공에대한 이해를 넘어선 믿음과 확신으로 이끌어야함

단순구성 - 이야기가 선형적
복합구성 - 관계없어보이는 에피소드들이 결합되어 발전함
피카레스크식 구성 - 옴니버스인데 주인공한명은 계속 고정임
옴니버스식 구성 - 주인공까지 바뀜


arch plot : 자신의 목적을 위하여 주로 외부의 저항세력과 맞서 싸우는 주인공을 중심으로 이야기를 구성하는 것

mini plot : 아크플롯을 따르되, 일부 특징을 축소하거나 삭제하는 경우

anti plot : 플롯부수기

none plot : 변화가 없고 정체된 이야기(아무런 이야기도 하지않음)
